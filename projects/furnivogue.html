Furnivogue E-Commerce

Website E-Commerce Furnitur dan Dekorasi Interior Rumah dengan Antarmuka Minimalis Modern yang Interaktif.

Deskripsi Proyek

Furnivogue E-Commerce Website adalah proyek freelance untuk toko furnitur modern bergaya Nordik (Scandinavian). Fokus utama dari perancangan web ini adalah menciptakan pengalaman berbelanja furnitur online yang interaktif, di mana pembeli dapat menyesuaikan spesifikasi produk sebelum membelinya.

Web ini dilengkapi dengan pemilih warna kain, jenis kayu, dan variasi dimensi furnitur yang mengubah harga produk secara dinamis. Untuk memfasilitasi pengiriman furnitur berdimensi besar dan berat, backend Laravel diintegrasikan dengan API jasa pengiriman kargo lokal guna mengkalkulasikan ongkos kirim secara otomatis di halaman checkout.

Desain minimalis dan bersih menonjolkan keindahan tekstur furnitur, meningkatkan keinginan nasabah untuk berbelanja.

Fitur Utama
  • Dynamic Product Selector: Pemilih variasi warna kain sofa, ukuran meja, dan bahan kayu dengan update gambar dan harga otomatis.
  • Shipping Cost API Integration: Perhitungan ongkos kirim cargo berdasarkan bobot volume barang dan kota tujuan pembeli.
  • High-Quality Image Gallery: Galeri foto produk dari berbagai sudut pandang dengan fitur zoom responsif.
  • Dashboard Keuangan Penjualan: Visualisasi performa omset bulanan toko bagi administrator web.
