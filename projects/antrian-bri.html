Sistem Antrian BRI

Aplikasi Pemanggilan Antrean Real-time Multi-kategori Terintegrasi API Laravel untuk Mengoptimalkan Alur Layanan Nasabah Bank BRI.

Deskripsi Proyek

Sistem Antrian BRI dikembangkan untuk menjawab kebutuhan digitalisasi antrean di bank BUMN terkemuka guna mempercepat penyaluran layanan perbankan di kantor cabang. Sistem ini menyediakan antarmuka mandiri di depan pintu masuk (kios tiket) di mana nasabah memilih kategori kebutuhan layanan (Teller A/B, Customer Service, atau Pengaduan).

Dengan backend framework Laravel yang tangguh, data antrean disimpan secara efisien di MySQL. Setiap counter Teller/CS memiliki panel kontrol digital sendiri untuk memanggil nomor antrean berikutnya. Sistem dilengkapi dengan panggilan audio Text-to-Speech otomatis berbahasa Indonesia yang secara dinamis memanggil nasabah menuju loket tujuan.

Analisis backend juga mengukur durasi layanan per nasabah demi membantu supervisor melacak produktivitas pelayanan counter harian.

Fitur Utama
  • Kios Tiket Mandiri: UI sentuh ramah pengguna untuk mencetak nomor antrean nasabah berdasarkan jenis urusan perbankan.
  • Panel Loket Pemanggil: Antarmuka web bagi Teller dan CS untuk memanggil, mengulang panggilan, atau mengalihkan antrean nasabah ke loket lain.
  • Panggilan Suara TTS (Text-To-Speech): Integrasi Web Speech API untuk memanggil nomor antrean secara otomatis ("Nomor antrean A-12, silakan menuju Teller 1").
  • Dashboard Monitor Publik: Menampilkan nomor antrean yang sedang dilayani di setiap loket beserta tayangan informasi promosi bank.
