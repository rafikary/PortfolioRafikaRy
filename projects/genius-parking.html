Genius Parking System

Sistem Deteksi Slot Parkir Rest Area Berbasis Computer Vision dan IoT - Kolaborasi Riset Matching Fund Dikti Ristek Bersama PT JASA MARGA PERSERO TBK.

Deskripsi Proyek

Genius Parking Space Detection System merupakan proyek riset terapan skala nasional hasil hibah Matching Fund Kedaireka 2023 dari Direktorat Jenderal Pendidikan Tinggi (DIKTI) bekerjasama dengan BUMN pengelola jalan tol, PT Jasa Marga Persero Tbk. Jakarta.

Tujuan utama proyek ini adalah mendigitalisasi pemantauan ketersediaan slot parkir di rest area jalan tol guna mencegah penumpukan kendaraan. Sistem menggunakan kamera CCTV yang dipasang di rest area untuk menangkap gambar secara real-time. Gambar tersebut diproses menggunakan algoritma deep learning pendeteksi objek (seperti YOLO dan OpenCV) untuk menentukan slot mana saja yang telah terisi (occupied) atau masih kosong (vacant).

Data status parkir tersebut kemudian disinkronisasikan ke server pusat melalui web API, yang selanjutnya dikirim ke signage digital fisik di depan gerbang rest area dan aplikasi web pantau admin.

Fitur Utama
  • Deteksi Parkir Real-time: Computer vision untuk mendeteksi mobil pada garis parkir di bawah berbagai kondisi cuaca dan pencahayaan.
  • Integrasi Papan LED Fisik: Mengirimkan sinyal ketersediaan slot sisa secara langsung ke papan digital fisik di area gerbang masuk rest area.
  • Admin Web Dashboard: Peta visual layout parkir interaktif yang memberikan status slot merah (terisi) / hijau (kosong) bagi manajemen rest area.
  • Analisis Densitas Rest Area: Rekaman log masuk-keluar untuk mengidentifikasi jam sibuk rest area tol.
