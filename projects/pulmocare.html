PulmoCare

Platform Diagnosa Awal Tingkat Risiko Kanker Paru-Paru Menggunakan Algoritma Klasifikasi Machine Learning Berdasarkan Gejala dan Gaya Hidup.

Deskripsi Proyek

PulmoCare adalah platform skrining kesehatan cerdas berbasis web yang dirancang untuk mengevaluasi tingkat risiko kanker paru-paru pada pasien. Proyek ini memproses dan memvisualisasikan data survei gejala dari dataset medis, mendeteksi korelasi kebiasaan sehari-hari dengan probabilitas timbulnya kanker.

Dengan memproses data seperti riwayat konsumsi rokok, paparan polusi, pola makan, tingkat kelelahan, sesak napas, hingga batuk berdarah, model klasifikasi machine learning memproses skor probabilitas secara real-time dan mengelompokkan risiko ke dalam kategori Rendah, Sedang, atau Tinggi.

Platform ini mempermudah integrasi model prediktif Python menggunakan REST API Flask agar dapat dipanggil dengan cepat dari frontend PHP/HTML.

Fitur Utama
  • Survei Gejala Komprehensif: Form pendaftaran gejala berbobot klinis yang mudah dipahami orang awam.
  • Multi-Model Predictor: Integrasi model Python (Flask) untuk inference instan.
  • Visualisasi Insight: Chart perbandingan tingkat risiko berdasarkan kebiasaan merokok dan usia untuk edukasi kesehatan.
  • Rekomendasi Tindakan: Arahan praktis pencegahan gaya hidup dan integrasi kontak medis.
